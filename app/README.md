# TMG Website

## Commands

- `npm install` - install dependencies
- `npm start` - start the local server
- `npm run build` - builds the artifacts - exact same command that will run automatically on commit
- `npm run lint` - prints all linting errors
- `npm run format` - fixes all linting errors
- `npx swa start` - runs the localserver in an Azure Static Web App emulator for login

Instead start up both functions and website with the readme in the parent directory.
